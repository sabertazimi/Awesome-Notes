Bug List

  • Bug List
    • Basic Bug
    • C Bug
    • 低概率/偶发性 Bug

1. Basic Bug

  • 必须进行输入验证 - 永远不要相信用户输入
  • 永不使用未经验证的数值的长度或大小
  • 必须返回正确的错误状态
  • 注意(隐式)类型转换

2. C Bug

  • 栈缓冲区溢出
  • 空指针解引用
  • (隐式)类型转换
  • GOT 覆写(Global Offset Table)

3. 低概率/偶发性 Bug

  • 多进程完全异步编程的复杂性
  • 逐渐地内存泄漏
